# Task.
- 차량 유리 손상 이미지 데이터셋을 기반으로 하여, 인공지능을 활용해 Repair, Replace 판단을 자동화하는 모델을 개발.
- 즉, 이진분류 분류 모델 개발.

# Problem Definition.
## Data Distribution. (Overfitting 우려)
- 데이터셋의 전체 크기가 2000장 정도로 적음.
- 차량 유리 손상 이미지 특성상 손상 위치 식별 목적으로 손상 주변에 Marking을 표시함.
- 다만, 모든 이미지에 Marking이 표시되어 있는 것은 아님.
- Repair의 경우 전체 이미지의 9.7%, Replace의 경우 전체 이미지의 54.5%가 Marking을 포함함.
- 즉, 이러한 불균형한 Marked Image의 비율로 인해 모델이 손상이 아닌 Marking을 기반으로 판단할 가능성이 존재함. Ex) Marking이 존재하면 Replace, 존재하지 않으면 Repair로 분류.

## Attention Misalignment.
- 차량 유리 손상 이미지 특성상 손상 주변에 손상과 관련없는 배경이 다수 존재함. Ex) 사람의 손, 차량의 부품 등.
- 이러한 배경이 모델의 Attention을 분산시켜, 손상 부위에 집중하지 못하게 함.

위 두 가지 문제점의 공통점은 모델이 손상 부위에 집중하지 못하게 만들며, 이는 곧 모델의 결과를 신뢰할 수 없도록 한다는 것임.

# Method.
## Architecture.
- Architecture는 두 가지의 Pre-hoc 기법을 활용함.
- 첫 번째는 Generative Annotation Cleaning (GAC) 기법임. Gemini-2.5-flash Image 모델을 활용한 instruction-guided generative 기법임. 이는 Gemini API를 호출하고, 이미지를 입력하여 프롬프트를 활용해 주위 배경은 그대로 두고, 손상 주위에 있는 Marking만 제거해달라고 하는 것임. 이를 통해, 이미지 주위에 있는 Marking을 제거함.
- 두 번째는 YOLO-CRop 기법임. YOLO 모델을 활용한 object detection 기법임. 이는 YOLO 모델을 활용하여 손상 부위를 탐지하고, 손상 부위를 중심으로 이미지를 크롭함. 이를 통해, 이미지 주위에 있는 Marking과 손상과 관련없는 배경을 제거함.
- 위 두 가지 단계를 거쳐 만들어진 데이터셋을 CNN 모델에 학습시켜, 최종 Repair, Replace 분류를 진행함.

## Approach.
- YOLO 모델의 경우, YOLO v2/v4/v5/v8을 활용하며 다양한 모델을 사용하여 성능을 비교함.
- CNN 모델의 경우, VGG16, ResNet152, MobileNet v1/v2/v3 Large을 활용하며 다양한 모델을 사용하여 성능을 비교함.
- 실험의 경우, Original / YOLO-Crop (Original에 YOLO-Crop만 적용) / GAC (Original에 GAC만 적용) / Hybrid(Original에 YOLO + GAC 적용) Dataset 네 가지 경우에 대해 성능을 비교함.

# Experiments.
- 첫 번째 실험으로 GAC의 성능을 검증하기 위해 Self-Supervised Protocol을 설계함. GAC의 정확한 성능을 측정하기 위하여 Marking이 없는 이미지를 GT로 설정하고, 이 이미지에 Marking을 강제로 생성함. 이후, GAC를 거쳐 Marking이 제거된 이미지를 얻어냄. 이후, GT 이미지에 JPEG 압축, Blur 기법을 적용하여 GT와 차이가 거의 없는 이미지를 얻어내고, GT 이미지에 다른 손상 이미지와 결합하여 완전히 손상이 바뀌었을 때 이미지를 얻어냄. 마지막으로, GT 이미지와 아무런 관계가 없는 Out-of-Distribution 이미지를 추가함. 이후, GT와 각각 Anchor의 손상 부위만을 크롭한 후, GT를 기준으로 각각 Anchor와의 Feature Map Distance, Edge L1, SSIM, PSNR을 비교함.
- 두 번째 실험으로 YOLO-Crop의 필요성을 검증함. 손상 주위에 Marking이 있는 이미지를 기준으로 하여, 손상 주위를 모두 Blur 처리한 이미지와 손상 주위를 Black-out 시킨 이미지와 GAC를 통해 Marking이 제거된 이미지와 YOLO-Crop을 통해 손상부위만 추출된 이미지를 얻어냄. 이후, 각 데이터셋을 CNN에 학습시킨 후 결과를 비교함. 이는 Marking을 없애면서 손상과 관련없는 배경을 제거하는 여러 기법들과 비교하며 우리가 왜 YOLO-Crop을 채택했는지에 대한 설득력을 높임.
- 세 번째 실험으로 YOLO-Crop의 Attention Alignment 성능을 검증함. Original Dataset과 Original Dataset에 YOLO-Crop을 적용하여 얻어낸 YOLO-Crop Dataset을 CNN 모델에 학습시킨 후 Grad-CAM 시각화 결과를 비교함. 이를 통해, YOLO-Crop이 Attention Alignment 성능을 향상시켰는지 검증함. 또한, GAC Dataset과 Hybrid Dataset에 대해서도 같은 과정을 수행하여, Attention Alignment 성능이 Marking의 유무와는 관계가 없음을 검증함.

# Results.
- 첫 번째 실험을 통해, GAC의 성능이 JPEG, Blur 기법같이 GT와 매우 유사한 이미지와 거의 비슷한 성능을 보이며, 다른 손상으로 뒤바뀐 Mix Anchor보다 훨씬 더 뛰어난 성능을 입증함. 또한, Edge L1과 Feature Map Distance를 각각 x,y 축으로 설정하여 Scatter Plot을 그렸을 때, 우리의 방법론은 JPEG, Blur와 거의 같은 군집군을 가짐.
- 두 번째 실험을 통해, GAC / Blur / Black-out / YOLO-Crop 순으로 성능이 향상됨을 입증함. 특히, 다른 기법들의 경우 정확도가 90%가 넘지 않는 반면, YOLO-Crop의 경우 97.46%로 다른 방법론들에 비해 월등히 뛰어난 성능을 보임.
- 세 번째 실험을 통해, YOLO-Crop이 Attention Alignment 성능을 향상시켰음을 입증함. 또한, GAC Dataset과 Hybrid Dataset에 대해서도 같은 결과를 보임. 즉, YOLO-Crop을 통해 손상과 관련 없는 배경 부위를 삭제하는 것이 모델의 신뢰성을 향상시킨다는 것을 확인함.
- Original / YOLO-Crop / GAC / Hybrid 데이터셋에 대한 정확도 성능을 비교함. 각 데이터셋을 CNN 모델에 입력하여 학습시키고, 분류 성능을 확인함. 그 결과, Original이 가장 높은 성능을 보이나 이는 Marking을 기반으로 판단한 Overfitting의 결과이며, 이는 신뢰할 수 없다고 판단함. 그렇다면, GAC를 통해 생성된 데이터셋을 Fair Baseline으로 판단하며, 그 경우 Hybrid 데이터셋이 GAC 데이터셋보다 약 4.7%의 정확도 향상을 보임.
- 또한, 모든 코드를 클린코드 기반으로 하여 MLOps를 구현함. 모든 코드는 Github에 공유되어 있으며, Streamlit을 활용하여 웹 데모를 만들어 Hugging Face Spaces로 배포함.
- 결과적으로, 교내 공모전 2회 수상 및 2024 KSC 게재, 현재는 SCI급 저널에 투고하고 결과를 기다리는 중임.
